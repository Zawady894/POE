experience

Attachment log

HARDWARE MAINTENANCE

Desktop diagnostics & repair

Opened up desktop units to inspect motherboards, RAM slots, and power connectors — diagnosing faults, reseating components, and testing hard drives to get machines back in working order.

NETWORKING

Network configuration & testing

Used ipconfig /all to inspect network adapter configuration and ran ping tests between hosts to verify connectivity and diagnose latency and packet loss.

SYSTEMS MONITORING

Performance checks

Reviewed CPU, memory, and disk usage through Task Manager and system tools to assess machine performance and flag units that needed upgrades or cleanup.

DATA & REPORTING

Spreadsheet analysis & business proposals

Built Excel sheets to track sales, cost of goods, and monthly profit, calculated profit percentages with formulas, and visualized trends with charts. Drafted a management proposal on strategies to improve profit margins.
